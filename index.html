Projects

OCR-TO-FORMATTER

This is a react web app that helps you identify text in your images using Tesseract.js. It runs completely in browser and also shows example of how to use the tesseract.js in react.

Link Noter

This is a react native app that helps you store links. In this world of articles, sometimes you may want to revisit an interesting article that you read once and lost in browser bookmarks. This is a perfect app for your design. It not only just stores the links but also gives the preview and also let you scroll through the links you noted. This is build using expo.

JSON-WORLD

Json World is a single page solution for all of your json needs. It has all basic functionalities like formating, escapping etc.It also comes with new features like uuid generator and also uuid-json replacer. It replaces all the uuid in the json payload with new ones intelligently.
